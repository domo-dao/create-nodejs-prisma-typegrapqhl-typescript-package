# Domo Starter for NodeJS, Typescript, Prisma - Type Grapqhl

It includes features for:

- Auth0 authentication
- Graphql Subscriptions
- Triggers on Operations
- Express compatible

## Main dependencies

- Node 20.9.0
- npm 10.1.0
- TypeScript 5.4.3
- Nodemon 3.1.0

## Run

1. Install

   ```bash
    npm i --legacy-peer-deps
   ```

2. Generate types

   ```bash
    npx prisma generate
   ```

3. Run

   ```bash
    npm run dev
   ```

## Production run

1. Build

   ```bash
    npm run build
   ```

2. Run

   ```bash
    npm run start
   ```
