# Insightt Portal

## Server

Local server
```
npm start
```

Development
```
npm run dev
```

Staging
```
npm run staging
```

master
```
npm run master
```

## Cron
Local server
```
npm run cron:start
```

Development
```
npm run cron:dev
```

Staging
```
npm run cron:staging
```

master
```
npm run cron:master
```
